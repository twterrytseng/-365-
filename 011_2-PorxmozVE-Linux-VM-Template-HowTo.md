# 開始製作Template

步驟一：開始複製 (Clone)

    找到 Template： 在 Proxmox VE 介面的左側資源樹中，找到您剛剛轉換好的 Template（它的圖示會是一個藍色的文件或紙張圖案）。

    開啟複製選項： 在該 Template 上點擊 右鍵，選擇 Clone。

    設定新 VM ID 和名稱：

        New VM ID: 填入新 VM 的 ID (例如：102)。

        Name (名稱): 填入新 VM 的 Hostname (例如：rocky-webserver-01)。
